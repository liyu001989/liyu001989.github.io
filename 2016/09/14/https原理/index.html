http 请求流程

  • 客户端发起请求 http://blog.lyyw.info/foo/bar
  • 客户端请求 DNS 服务器得到IP 133.130.121.201
  • 客户端封装 HTTP 请求数据包
  • 客户端根据 TCP 协议，分装为 TCP 数据包
  • 双方通过 TCP（三次握手）与服务器应用建立连接
  • TCP 将数据分割为 IP 包，数据帧，bit 流，传送数据到服务器
  • 服务器硬件首先收到bit流，然后转换为 IP 数据包
  • 服务器端，通过 TCP 协议将一个个的 IP 包组装为 TCP 数据包
  • 服务端段，通过 HTTP 协议将 TCP 数据包，组装为 HTTP 数据包
  • 服务端理解根据请求，返回数据包，已同样的流程返回数据到客户端，然后关闭 TCP 链接
